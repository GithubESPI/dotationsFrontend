# Installation des dépendances manquantes

Pour que le formulaire d'allocation fonctionne correctement, vous devez installer les dépendances suivantes :

```bash
npm install react-hook-form @hookform/resolvers
```

Ces packages sont nécessaires pour :
- `react-hook-form` : Gestion des formulaires avec validation
- `@hookform/resolvers` : Résolveur Zod pour la validation avec react-hook-form

Après l'installation, le formulaire d'allocation sera pleinement fonctionnel.
